# Welcome

Hi, I’m Mark Olmscheid. I’m currently studying computer science through CU Boulder. This is a public site where I document my learning, share projects, and explore technical topics I find interesting.

## Current Focus

I’m currently working on a graph algorithm learning series and visualization tool. The goal of the project is to create an interactive graph algorithm visualizer that explains how various graph algorithms work, including BFS, DFS, Dijkstra’s algorithm, and topological sort. I’m building the project using Python and libraries that support visualization and interactivity.

You can follow my progress through the blog, where I’ll post weekly updates, project notes, lessons learned, and reflections on the development process.
