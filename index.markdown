# Welcome

Hi, I’m Mark Olmscheid. I’m currently studying Applied Computer Science through CU Boulder while building projects that I find interesting. This is a public site to document my learning, share projects, and explore technical topics that I find interesting.

## Current Focus

I’m currently working on a graph algorithm learning series and visualization tool. The goal of the project is to create an interactive graph algorithm visualizer that can be used to explain how various graph algorithms work, including BFS, DFS, Dijkstra’s Algorithm, and Topological Sort. I’m building the project using Python and various libraries for visualization and interactivity.

You can follow my progress through the blog, where I’ll post weekly updates, project notes, lessons learned, and reflections on the development process.
